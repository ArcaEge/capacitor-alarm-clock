# Assets

3D PCB renders were generated with KiCAD CLI, using the `render` command.

Banner and poster designs were designed in Figma, [available here](https://www.figma.com/design/jmvWcAlcJrcGd9PXcn2myq/Capacitor-alarm-clock). The colour palette used is from [here](https://colordesigner.io/color-palette-builder#703030-2F343B-7E827A-E3CDA4-C77966).
